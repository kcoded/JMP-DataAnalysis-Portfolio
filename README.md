## About Me

I'm a Process and Manufacturing Engineer with a background in semiconductor process engineering. I'm currently completing the JMP Visual Six Sigma certification, and this repository documents that work by applying statistical tools to real manufacturing and quality problems rather than just working through slides.

My core competencies include exploratory data analysis, statistical process control, design of experiments, measurement systems analysis, process capability, DMAIC, regression, and hypothesis testing.

## References and Course Attribution

This work was completed as part of the JMP Visual Six Sigma online statistics course, available at the [JMP Visual Six Sigma online statistics course](https://www.jmp.com/en/online-statistics-course) page, using JMP 17 Pro. The datasets used throughout the portfolio were provided as course materials and include Measles.jmp, Airline Delays EDA.jmp, Particles 2.jmp, Reactor 20 Custom.jmp, Custom CCD 3 Factors.jmp, and Anodize.jmp, along with additional datasets used in the Quality Methods module (CrisisTeamData, MetalPartsAll, MetalPartsAfter, Diameter SPC, Impurity, PartFlatness, MSA_MFI_Initial, and the Area MSA exercise data).

All analyses, interpretations, and visualizations in this repository were performed independently using the course data and JMP software. The findings and conclusions are my own work.
